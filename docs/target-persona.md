# NEET Quest - ターゲットペルソナ

## プライマリペルソナ

### プロフィール

| 項目 | 内容 |
|------|------|
| 名前 | 田中 ユウキ（仮） |
| 年齢 | 24歳 |
| 性別 | 男性 |
| 居住 | 実家（関東近郊） |
| 状況 | 大学中退後、2年間ほぼ外出せず生活 |
| デバイス | スマホ（iPhone）、自室のPC |

### 心理・行動特性

- 社会不安を抱えており、外出や人との会話にストレスを感じる
- 自分の状況を自虐的に笑えるユーモアがある
- 深夜にブラウザゲームやYouTubeを長時間消費する
- 「変わりたいけど動けない」というジレンマを常に感じている
- 重いテーマを深刻に説教されるのは嫌い。軽いノリで触れてほしい

### ゲームに求めるもの

- **共感**: 自分の日常や心理状態がゲーム内に反映されている安心感
- **ユーモア**: 「不安スライム」「怠惰ゴーレム」など敵名で笑える
- **達成感**: 小さなクエストクリアで「自分にもできた」と思える体験
- **手軽さ**: インストール不要、ブラウザで即プレイ、1プレイ10-30分

### 利用シーン

- 深夜2時、布団の中でスマホからプレイ
- Twitterで「ニートRPG」がバズっているのを見て流入
- 1日1クエストずつ、数日かけてクリア

## セカンダリペルソナ

### プロフィール

| 項目 | 内容 |
|------|------|
| 名前 | 鈴木 サキ（仮） |
| 年齢 | 28歳 |
| 性別 | 女性 |
| 職業 | 会社員（事務職） |
| 趣味 | インディーゲーム、RPGツクール作品 |

### 心理・行動特性

- 社会問題に関心があり、ゲームを通じた表現に共感する
- RPGが好きで、ストーリー重視のフリーゲームをよく探している
- SNSで気になったゲームを即試すフットワークの軽さがある
- 面白いと思ったらスクショ付きで感想を投稿する

### ゲームに求めるもの

- **物語性**: 主人公の成長を通じた社会復帰のストーリー
- **風刺とユーモア**: 「ハローワーク」「面接ダンジョン」などの社会風刺
- **完成度**: 短くてもまとまったゲーム体験
- **共有性**: 感想やスクショをSNSに投稿したくなる要素

## ペルソナから導くデザイン方針

| ペルソナの特性 | デザインへの反映 |
|----------------|------------------|
| 深夜にプレイ | ダークテーマUI、目に優しい配色 |
| スマホ利用が多い | モバイルファースト、タップ操作対応 |
| 長時間集中が難しい | 1クエスト5-10分で完結する設計 |
| 自虐ユーモアが好き | 敵名・アイテム名・テキストに自虐ネタ |
| 説教されたくない | 教訓は押し付けず、プレイヤーが自然に感じる設計 |
| SNS拡散が見込める | シェアボタン、スクショ映えする場面演出 |
| インストール不要 | PWA対応、URLだけでプレイ開始 |
