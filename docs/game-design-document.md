# NEET Quest - ゲームデザインドキュメント (GDD)

## 概要

| 項目 | 内容 |
|------|------|
| タイトル | NEET Quest ~社会復帰への冒険~ |
| ジャンル | ターン制RPG |
| プラットフォーム | Webブラウザ（モバイルレスポンシブ対応） |
| アートスタイル | 絵文字ベースのタイルマップ、ダークテーマUI |
| エンジン | HTML5 Canvas / DOM |

## コンセプト

引きこもり（ニート）の主人公が、心理的な敵と戦いながら社会復帰を目指すRPG。不安・怠惰・自己否定といった心の壁をモンスターとして具現化し、クエストを通じて少しずつ外の世界へ踏み出していく。

## コアゲームプレイループ

```
マップ探索 → NPC会話 → クエスト受注 → 敵とエンカウント → 戦闘
    ↑                                                    ↓
    ←── レベルアップ ← 報酬獲得 ← 勝利 ← ターン制バトル ←┘
    ↓
社会性ポイント上昇 → 新エリア解放 → ストーリー進行
```

## ゲームフロー

```
タイトル画面
  ├─ はじめから
  └─ つづきから
      ↓
  ゲーム画面（マップ探索）
  ├─ メニュー（ステータス / 装備 / アイテム / クエスト / セーブ）
  ├─ NPC会話 → クエスト受注/完了
  ├─ ショップ → アイテム売買
  └─ エンカウント → バトル画面
                      ├─ 勝利 → 経験値・ゴールド獲得
                      ├─ 敗北 → 自室に戻る（ゴールド半減）
                      └─ 逃走
```

## マップ一覧

| マップID | マップ名 | 解放条件 | 概要 |
|----------|----------|----------|------|
| `my_room` | 自分の部屋 | 初期 | チュートリアル。ベッド、PC、ゴミの山。セーブポイント |
| `home` | 自宅 | クエスト「部屋を出る」完了 | 母親NPC、キッチン、玄関 |
| `town` | 近所の通り | 社会性Pt 5以上 | 商店街、通行人NPC、公園 |
| `convenience_store` | コンビニ | 社会性Pt 10以上 | 初めてのバイト先候補、店長NPC |
| `hello_work` | ハローワーク | 社会性Pt 20以上 | 求人検索、相談員NPC |
| `interview_dungeon` | 面接ダンジョン | クエスト「履歴書を書く」完了 | ボスフロア。面接官がラスボス |

## バトルシステム

### 基本ルール
- ターン制（素早さ順で行動）
- コマンド選択式

### コマンド

| コマンド | 説明 |
|----------|------|
| こうげき | 通常攻撃。ATK依存ダメージ |
| スキル | MP消費で特殊技を使用 |
| アイテム | 回復アイテム等を使用 |
| にげる | 逃走を試みる（ボス戦は不可） |

### ダメージ計算

```
物理ダメージ = ATK * 2 - DEF + random(-3, 3)
スキルダメージ = スキル倍率 * ATK - DEF / 2
```

### スキル例

| スキル名 | MP | 効果 | 習得Lv |
|----------|----|------|--------|
| 自己暗示 | 3 | ATK 1.5倍で攻撃 | 1 |
| 深呼吸 | 4 | HP 20回復 | 3 |
| 正論パンチ | 6 | ATK 2.0倍で攻撃 | 5 |
| ポジティブシンキング | 8 | ATK+DEF 3ターン上昇 | 8 |
| 社会の窓を開く | 15 | 全体に大ダメージ | 12 |

## 成長システム

### ステータス

| パラメータ | 略称 | 説明 |
|------------|------|------|
| HP | HP | 体力。0で戦闘不能 |
| MP | MP | 精神力。スキル使用に消費 |
| ATK | ATK | 攻撃力 |
| DEF | DEF | 防御力 |
| SPD | SPD | 素早さ。行動順に影響 |

### レベルアップ時の成長値

- HP: +8
- MP: +4
- ATK: +2
- DEF: +2
- SPD: +1

### 社会性ポイント

クエスト完了やNPC会話で獲得。新エリア解放やストーリー進行のトリガーとなる。戦闘では変動しない。

## クエスト例

| クエスト名 | 発生場所 | 内容 | 報酬 |
|------------|----------|------|------|
| 部屋を出る | 自分の部屋 | 部屋のゴミを片付ける | 社会性Pt +3, 50G |
| 母の頼み | 自宅 | コンビニで牛乳を買ってくる | 社会性Pt +5, 100G |
| 初めてのバイト | コンビニ | レジ打ちミニゲーム3回クリア | 社会性Pt +10, 500G |
| 履歴書を書く | ハローワーク | 必要アイテム収集 | 社会性Pt +15, 面接ダンジョン解放 |
| 最終面接 | 面接ダンジョン | ボス「圧迫面接官」を倒す | エンディング |
