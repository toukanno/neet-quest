# ニートクエスト

ニートが勇者として社会復帰するRPG — ブラウザで動作するWebアプリケーション

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

## ビルド

```bash
npm run build
npm run preview
```

## 技術スタック

- React 19 + TypeScript
- Vite (ビルドツール)
- Zustand (状態管理)
- React Router (画面遷移)
- ESLint (コード品質)

## 機能

- タイトル画面 / ニューゲーム / つづきから
- グリッドベースのマップ探索 (キーボード WASD / 矢印キー / タッチ D-pad)
- NPC会話システム (選択肢分岐あり)
- ターン制バトルシステム (通常攻撃 / スキル / アイテム / 逃走)
- レベルアップ・経験値システム
- インベントリ管理 (消費アイテム / 装備)
- クエストシステム (メイン / サブクエスト)
- セーブ / ロード (localStorage, 3スロット)
- 社会復帰度メーター
- レスポンシブ対応 (モバイル / デスクトップ)

## マップ

- 自分の部屋 → 自宅リビング → 近所の街
- コンビニ / ハローワーク / 面接ダンジョン
